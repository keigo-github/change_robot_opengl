# change_robot_opengl

opene GLを用いたアニメーションの作成
マウスによる視点操作、縮小拡大が可能
'e'をコマンドすると、15度ずつブロックが回転し、180度回転した時点でロボットが変形し人型になる。
床面に草原を表現するテクスチャの割り当てを行った。
